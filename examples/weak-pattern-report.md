## Recommended drill order

### 1. Dynamic Programming (highest ROI — appears in most final rounds)

Not started or barely started:
- Climbing Stairs (Easy) — start here if DP is new: [NeetCode](https://neetcode.io/problems/climbing-stairs) · [AlgoMonster](https://algo.monster/practice)
- House Robber (Medium): [NeetCode](https://neetcode.io/problems/house-robber)
- Coin Change (Medium): [NeetCode](https://neetcode.io/problems/coin-change)
- Word Break (Medium): [NeetCode](https://neetcode.io/problems/word-break)
- Longest Increasing Subsequence (Medium): [NeetCode](https://neetcode.io/problems/longest-increasing-subsequence)

Cue: **optimal substructure** — count ways, minimize/maximize a value, or find longest subsequence.
Trap: not defining the DP state clearly before writing the recurrence.
